# Bellabeat Case Study – Google Data Analytics Capstone

This project applies the full data analysis process to explore smart device usage data from Bellabeat, a high-tech wellness company. The goal is to uncover insights that can inform marketing strategies.

## Tools Used
- R (or Excel/Sheets if applicable)
- SQL (learning phase)
- Tableau (or Power BI if used)

## Steps Followed
1. **Ask** – Define business task and key questions
2. **Prepare** – Review and clean FitBit data
3. **Process** – Handle missing values, format columns
4. **Analyze** – Explore trends in activity, sleep, and calories
5. **Share** – Visualize findings in Tableau/Power BI
6. **Act** – Recommend marketing strategies based on insights
